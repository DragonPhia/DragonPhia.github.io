Admin Bulk Upload

Select a JSON, CSV, or TXT file to upload product data to the database.



Example File Format

The uploaded file should be in one of the following formats:

JSON Format:

    [
        {
            "name": "Product 1",
            "price": 15.99,
            "description": "Description of product 1",
            "image": "product1.jpg"
        },
        {
            "name": "Product 2",
            "price": 29.99,
            "description": "Description of product 2",
            "image": "product2.jpg"
        }
    ]

CSV Format:

    name,price,description,image
    Product 1,19.99,Description of product 1,product1.jpg
    Product 2,29.99,Description of product 2,product2.jpg
